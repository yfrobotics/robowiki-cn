## 机器人学综合课程

3. [Stanford CS223A - Introduction to Robotics (YouTube)](https://www.youtube.com/playlist?list=PL65CC0384A1798ADF), Oussama Khatib — 斯坦福大学机器人学导论课程，系统讲解运动学、动力学和控制
4. [MIT 6.832 - Underactuated Robotics (YouTube)](https://www.youtube.com/channel/UChfUOAhz7ynELF-s_1LPpWg), Russ Tedrake — MIT 欠驱动机器人课程，涵盖动态系统、轨迹优化和运动规划
5. [Modern Robotics (YouTube)](https://www.youtube.com/playlist?list=PLggLP4f-rq02vX0OQQ5vrCxbJrzamYDfx), Kevin Lynch — 配合《Modern Robotics》教材的完整视频课程，采用旋量理论体系

## 控制系统（Control Systems）

6. [Brian Douglas - Control Systems Lectures (YouTube)](https://www.youtube.com/user/ControlLectures) — 控制系统入门的优秀视频系列，以直观清晰的方式讲解 PID 控制、波特图、根轨迹等基础概念
7. [Steve Brunton - Control Bootcamp (YouTube)](https://www.youtube.com/playlist?list=PLMrJAkhIeNNR20Mz-VpzgfQs5zrYi085m) — 华盛顿大学教授讲解的控制理论短课程，覆盖状态空间方法和最优控制
8. [MATLAB Tech Talks - Control Systems (YouTube)](https://www.youtube.com/playlist?list=PLn8PRpmsu08pFBqgd_6Bi7msgkWFKL33b) — MathWorks 官方的控制系统技术讲座系列

## 机器学习与深度学习（Machine Learning & Deep Learning）

9. [Stanford CS229 - Machine Learning (YouTube)](https://www.youtube.com/playlist?list=PLoROMvodv4rMiGQp3WXShtMGgzqpfVfbU), Andrew Ng — 吴恩达经典机器学习课程，系统讲解监督学习、无监督学习和强化学习基础
10. [Stanford CS231n - Convolutional Neural Networks for Visual Recognition (YouTube)](https://www.youtube.com/playlist?list=PL3FW7Lu3i5JvHM8ljYj-zLfQRF3EO8sYv) — 斯坦福计算机视觉和深度学习课程，是学习卷积神经网络的经典入门课程
11. [DeepMind x UCL - Reinforcement Learning Lecture Series (YouTube)](https://www.youtube.com/playlist?list=PLqYmG7hTraZDVH599EItlEWsUOsJbAodm) — DeepMind 与伦敦大学学院合作的强化学习课程，由 David Silver 等人主讲
12. [UC Berkeley CS285 - Deep Reinforcement Learning (YouTube)](https://www.youtube.com/playlist?list=PL_iWQOsE6TfVYGEGiAOMaOzzv41Jfm_Ps), Sergey Levine — 深度强化学习前沿课程，涵盖模型学习、离线 RL 和机器人学习

## 计算机视觉（Computer Vision）

13. [First Principles of Computer Vision (YouTube)](https://www.youtube.com/channel/UCf0WB91t8Ber7GNsyag), Shree Nayar — 哥伦比亚大学教授讲解计算机视觉的基本原理，从成像模型到特征检测
14. [Cyrill Stachniss - Photogrammetry and Computer Vision (YouTube)](https://www.youtube.com/c/CyrillStachniss) — 波恩大学教授的计算机视觉和 SLAM 系列课程，内容详实
15. [Multiple View Geometry (YouTube)](https://www.youtube.com/playlist?list=PLTBdjV_4f-EJn6udZ34tht9EVIW7lbeo4), Daniel Cremers — TU Munich 教授讲解多视图几何，是学习视觉 SLAM 的理论基础

## ROS 机器人开发

16. [ROS Tutorials - ROS Wiki](http://wiki.ros.org/ROS/Tutorials) — ROS 官方入门教程，涵盖话题、服务、参数服务器等基本概念（非视频，但为入门必读）
17. [ROS 2 Tutorials (YouTube)](https://www.youtube.com/playlist?list=PLNw2RD-1J5YZge0JNaWWpqo21ml8YOQES) — ROS 2 入门视频教程系列
18. [The Construct - ROS Learning (YouTube)](https://www.youtube.com/c/TheConstruct) — 提供 ROS 和 ROS 2 的系统化学习视频，涵盖导航、操作和仿真

## SLAM 与自主导航

19. [Cyrill Stachniss - SLAM Course (YouTube)](https://www.youtube.com/playlist?list=PLgnQpQtFTOGQrZ4O5QzbIHgl3b1JHimN_) — SLAM 专题课程，从理论到实践系统讲解
20. [视觉SLAM十四讲 配套视频](https://www.bilibili.com/video/BV1Xv411s7Tk) — 配合高翔博士的《视觉SLAM十四讲》教材，适合中文学习者
