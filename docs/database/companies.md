# 机器人公司企业

## 面向工业领域的机器人
- **ABB** — 总部位于瑞士苏黎世，全球领先的工业自动化和机器人供应商，产品覆盖焊接、搬运、喷涂等多种应用场景
- **KUKA** — 德国工业机器人制造商，2017 年被中国美的集团收购，以橙色机械臂闻名，广泛应用于汽车制造业
- **FANUC** — 日本发那科公司，全球最大的工业机器人制造商之一，以高可靠性和数控系统（CNC）技术著称
- **Yaskawa（安川电机）** — 日本工业机器人四大家族之一，Motoman 系列机器人广泛应用于弧焊和搬运领域
- **Universal Robots** — 丹麦协作机器人（Collaborative Robot）先驱，其 UR 系列协作机械臂开创了人机协作的新模式
- **埃斯顿（Estun）** — 中国领先的工业机器人企业，产品线涵盖六轴机器人、SCARA 机器人及运动控制系统
- **新松机器人（SIASUN）** — 隶属于中国科学院，中国最大的机器人上市公司之一，产品包括工业机器人、移动机器人和特种机器人

## 面向服务机器人
- **SoftBank Robotics** — 日本软银旗下机器人公司，开发了 Pepper 和 NAO 人形机器人，主要面向商业接待和教育场景
- **iRobot** — 美国公司，Roomba 扫地机器人的制造商，是家用服务机器人领域的代表企业
- **科沃斯（Ecovacs）** — 中国领先的家用服务机器人企业，产品包括扫地机器人、擦窗机器人等
- **优必选（UBTECH）** — 中国人形机器人企业，产品涵盖教育机器人 Alpha 系列和大型人形机器人 Walker
- **达闼机器人（CloudMinds）** — 中国云端智能机器人公司，通过云端大脑为服务机器人提供智能能力

## 面向通用任务机器人
- **Boston Dynamics** — 美国机器人公司，以 Atlas 人形机器人和 Spot 四足机器人闻名，在动态运动控制领域处于全球领先地位
- **Agility Robotics** — 美国公司，研发双足行走机器人 Digit，面向物流搬运等通用任务场景
- **Figure AI** — 美国人形机器人初创公司，致力于开发通用人形机器人用于劳动力替代
- **Tesla（特斯拉）** — Optimus（擎天柱）人形机器人项目，目标是开发能在工厂和家庭中执行通用任务的人形机器人
- **1X Technologies** — 挪威机器人公司，开发 EVE 和 NEO 系列人形机器人，由 OpenAI 投资

## 面向自动驾驶
- **Waymo** — Alphabet（Google 母公司）旗下自动驾驶公司，运营商业化无人出租车（Robotaxi）服务
- **Uber** — 曾大力投入自动驾驶研发，后将自动驾驶部门 ATG 出售给 Aurora Innovation
- **百度** — 中国互联网巨头，旗下 Apollo 平台是全球最大的开放自动驾驶平台之一，已推出萝卜快跑（Apollo Go）无人出租车服务
- **Tesla** — 以 Autopilot 和 FSD（Full Self-Driving）系统著称，采用纯视觉方案进行端到端自动驾驶
- **Pony.ai** — 中美两地运营的自动驾驶公司，专注于 L4 级别自动驾驶技术
- **Aurora Innovation** — 美国自动驾驶公司，专注于自动驾驶卡车（Autonomous Trucking）物流运输
- **Momenta** — 中国自动驾驶公司，采用量产自动驾驶和完全无人驾驶双路线并行策略
- **小马智行（WeRide）** — 中国自动驾驶公司，在广州等城市运营自动驾驶出租车

## 面向医疗机器人
- **Intuitive Surgical** — 美国公司，达芬奇（da Vinci）手术机器人系统的制造商，是手术机器人领域的绝对领导者
- **Medtronic（美敦力）** — 全球医疗器械巨头，其 Hugo RAS 手术机器人系统正在拓展微创手术市场
- **天智航（Tinavi）** — 中国骨科手术机器人公司，天玑系列骨科手术导航定位机器人已获批临床应用
- **微创医疗机器人（MicroPort）** — 中国手术机器人公司，产品涵盖腔镜、骨科、血管介入等多个手术机器人领域

## 面向农业机器人
- **John Deere** — 美国农业机械巨头，正积极推进农业机器人和自动驾驶拖拉机技术
- **极飞科技（XAG）** — 中国农业无人机和农业机器人企业，产品用于农药喷洒、播种和农田测绘
- **大疆农业（DJI Agriculture）** — 大疆旗下农业无人机业务，T 系列植保无人机在全球市场占有率领先
- **Carbon Robotics** — 美国公司，开发基于激光的自主除草机器人

## 面向物流机器人
- **Amazon Robotics** — 原 Kiva Systems，被亚马逊收购后为其仓储物流提供大规模机器人分拣系统
- **Geek+（极智嘉）** — 中国智能物流机器人公司，提供仓储拣选、搬运和分拣机器人解决方案
- **快仓智能（Quicktron）** — 中国仓储物流机器人企业，由阿里巴巴投资
- **海柔创新（Hai Robotics）** — 中国 ACR（箱式仓储机器人）系统供应商，在自动化仓储领域具有创新性

## 面向空中机器人（无人机）
- **DJI（大疆）** — 中国深圳，全球消费级和行业级无人机市场的领导者
- **Skydio** — 美国无人机公司，以自主飞行和避障技术著称
- **Parrot** — 法国无人机制造商，面向消费级和行业应用市场

## 机器人核心算法
- **OpenAI** — 美国人工智能研究公司，在机器人学习、大语言模型等领域有深入研究，并投资人形机器人企业
- **DeepMind** — Alphabet 旗下人工智能实验室，在强化学习、蛋白质结构预测等领域取得突破性成果
- **NVIDIA** — 其 Isaac 平台为机器人提供 GPU 加速的仿真、感知和导航能力，Jetson 系列是机器人边缘计算的主流平台
