## 博士项目比较


### 北美博士项目特点

- **学制**：通常 5-7 年，包含课程学习和资格考试阶段
- **资助**：大部分学生获得全额资助（RA 或 TA）
- **灵活性**：入学后可以更换导师和研究方向
- **课程要求**：需要完成一定学分的课程
- **代表性项目**：CMU RI PhD、MIT EECS PhD、Stanford CS PhD、UMich Robotics PhD


### 欧洲博士项目特点

- **学制**：通常 3-4 年，纯研究导向
- **资助**：以项目制为主，博士生通常作为研究员受聘
- **入学要求**：通常需要硕士学位
- **独立性**：更强调独立研究能力
- **代表性项目**：ETH Zurich Doctoral Program、Max Planck 博士项目、EU Marie Curie ITN


### 亚太博士项目特点

- **学制**：通常 3-5 年
- **资助**：政府奖学金和大学助学金
- **语言**：日本和韩国部分项目可用英语完成
- **代表性项目**：东京大学 IST、KAIST 机器人学博士、新加坡 NTU 博士项目


### 博士项目选择建议

| 考量因素 | 说明 |
|---------|------|
| 导师匹配度 | 研究方向和指导风格是否匹配 |
| 毕业生走向 | 往届毕业生的就业和学术发展情况 |
| 实验条件 | 机器人平台和计算资源 |
| 发表文化 | 是否鼓励学生以第一作者发表顶会论文 |
| 地理位置 | 周边产业生态和生活成本 |
| 合作机会 | 实验室的产学合作网络 |
| 签证和政策 | 国际学生签证和留学政策的稳定性 |
