# NAO
NAO（发音和"Now"相近）是一个自主的可编程仿人机器人，由法国Aldebaran机器人公司研发。

2004年，NAO项目启动。2007年，NAO被索尼公司的AIBO机器狗击败，未能被RoboCup标准平台联赛(SPL)采用。在2008、2009年度的RoboCup比赛中，NAO被正式采用。NAOV3R则被用于2010年的RoboCup。

NAO具有两个不同定位的版本：学术版与家庭版。NAO家庭版面世较早，面向个人、家庭、服务行业，用于互动娱乐。NAO学术版面向大学和实验室，用于科学研究和教学。NAO学术版2011年推出市场。2010年10月，东京大学中村实验室购买了30台NAO机器人，希望将它们开发为实验室助理。从2011开始，NAO已经在全世界众多学术机构中被广泛使用，包括印度坎普尔理工学院。

2010年夏，NAO在上海世博会上以同步舞蹈动作惊艳四座，成为全球瞩目的焦点。2010年12月，NAO进行了一次令人惊异的单口相声表演。不久，NAO的下一代也发布，更加精巧的手臂和改进的马达成为突出特色。2011年12月，Aldebaran机器人公司发布了Nao Next Gen，具有更强健的软件，更快的CPU和高清摄像头。

## 技术规格

NAO 第六代（NAO6）的主要技术参数如下：

| 参数 | 规格 |
|------|------|
| 身高 | 57.4 cm |
| 体重 | 5.48 kg |
| 自由度（DOF） | 25 个（头部 2、手臂 2×5、手部 2×1、腿部 2×5、髋部 1） |
| 处理器 | Intel Atom E3845 四核 1.91 GHz |
| 内存 | 4 GB DDR3 RAM |
| 存储 | 32 GB eMMC |
| 电池 | 48.6 Wh 锂离子电池，续航约 60-90 分钟 |
| 操作系统 | NAOqi OS（基于 Linux） |

### 传感器

NAO 搭载了丰富的传感器系统：

- **视觉**：两个高清摄像头（OV5640），分辨率可达 2560×1920，分别位于前额和嘴部位置，支持不同视角
- **声音**：四个麦克风（位于头部），支持声源定位（Sound Source Localization）
- **触觉**：头部三个触摸传感器，手部和脚部各有触摸/压力传感器
- **惯性测量**：惯性测量单元（IMU），包含加速度计和陀螺仪
- **足底传感器**：每只脚底四个力敏电阻（FSR），用于检测压力分布和重心
- **红外**：两个红外发射器和接收器
- **声纳**：两个超声波传感器（Sonar），用于障碍物检测

## 编程与开发

### NAOqi SDK

NAOqi 是 NAO 机器人的核心软件框架（Framework），提供了一套模块化的 API 接口，涵盖运动控制、语音识别、视觉处理等功能模块。开发者可以通过 Python、C++ 等编程语言调用 NAOqi SDK 进行二次开发。

主要模块包括：

- **ALMotion**：运动控制模块，负责关节角度控制、步态行走、笛卡尔空间运动等
- **ALTextToSpeech**：语音合成模块，支持多语言文字转语音
- **ALSpeechRecognition**：语音识别模块，可识别预定义的词汇列表
- **ALVideoDevice**：视觉模块，获取摄像头图像数据
- **ALFaceDetection**：人脸检测模块
- **ALSonar**：声纳模块，用于距离测量
- **ALMemory**：共享内存模块，用于模块间数据传递

### Choregraphe

Choregraphe 是 Aldebaran 提供的图形化编程环境（Graphical Programming Environment），用户可以通过拖拽功能模块（Box）的方式为 NAO 编写行为程序，无需编写代码。Choregraphe 非常适合教育场景和快速原型开发，同时也支持嵌入 Python 脚本以实现更复杂的逻辑。

## 主要能力

### 语音交互（Speech Interaction）

NAO 支持多语言的语音合成（Text-to-Speech, TTS）和基础的语音识别功能。它能够与用户进行简单的对话交互，识别预设的关键词和短语，并通过内置扬声器进行语音输出。

### 视觉感知（Vision）

NAO 的双摄像头系统支持人脸检测与识别、物体识别、颜色检测、地标识别等视觉任务。结合 OpenCV 等计算机视觉库，开发者可以为 NAO 实现更高级的视觉功能。

### 自主行走（Locomotion）

NAO 具备双足步行能力，能够在平坦地面上自主行走、转弯、侧移，并具有一定的跌倒检测和自主起身能力。其步态算法基于 ZMP（Zero Moment Point）原理，保证行走过程中的动态平衡。

## 在教育和科研中的应用

NAO 是全球高校和科研机构中使用最广泛的人形机器人平台之一。其主要应用方向包括：

- **人机交互研究（HRI）**：NAO 的类人外观和丰富的传感器使其成为研究人机交互的理想平台
- **自闭症儿童辅助治疗**：多项研究表明，NAO 可用于辅助自闭症谱系障碍（ASD）儿童的社交技能训练
- **STEM 教育**：NAO 被广泛用于大学和中学的编程教学、机器人课程
- **多机器人协作**：多台 NAO 组成的团队可用于研究多智能体协调与协作
- **步态规划与控制**：NAO 是研究双足步行算法和平衡控制的重要实验平台

## RoboCup 标准平台联赛（SPL）

RoboCup 标准平台联赛（Standard Platform League, SPL）是 RoboCup 中唯一要求所有参赛队伍使用相同硬件平台的联赛。自 2008 年起，NAO 成为 SPL 的指定平台，取代了之前使用的索尼 AIBO 机器狗。

在 SPL 中，每支队伍使用数台 NAO 机器人进行自主足球比赛。机器人需要完全自主运行，包括视觉定位、球检测、路径规划、团队策略等，不允许任何人工干预。SPL 推动了自主机器人视觉、实时决策和多机器人协调等技术的发展。

## NAO 与 Pepper 的对比

2014 年，Aldebaran（后被软银收购并更名为 SoftBank Robotics）推出了 Pepper 机器人。NAO 与 Pepper 虽出自同一家族，但定位有所不同：

| 对比项 | NAO | Pepper |
|--------|-----|--------|
| 身高 | 57.4 cm | 120 cm |
| 自由度 | 25 DOF | 20 DOF |
| 移动方式 | 双足步行 | 轮式底盘 |
| 主要用途 | 教育与研究 | 商业接待与服务 |
| 显示屏 | 无 | 胸前 10.1 英寸平板 |
| 情感识别 | 基础 | 内置情感引擎 |

NAO 更适合需要双足步行和精细操作的学术研究场景，而 Pepper 更侧重于商业环境中的人机交互和信息展示。
