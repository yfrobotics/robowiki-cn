# 机器人实验室与研发机构 (国内)

以下收录中国大陆及港澳地区主要的机器人相关实验室与研发机构，涵盖高校实验室和科研院所。

- 中国 (China)
    - **清华大学自动化实验室** — 研究方向包括智能机器人、模式识别、控制理论与工程，在多机器人系统和无人驾驶领域有重要成果
    - **北京大学人工智能研究实验室** — 聚焦于机器学习、计算机视觉和自然语言处理等人工智能核心方向，为机器人智能化提供算法支撑
    - **哈尔滨工业大学机器人研究所** — 中国最早开展机器人研究的单位之一，在空间机器人、工业机器人和微纳机器人领域有深厚积累
    - **中国科学院沈阳自动化研究所** — 中国机器人事业的摇篮，研制了中国第一台工业机器人，在水下机器人、特种机器人和工业自动化领域处于国内领先地位
    - **北京航空航天大学机器人研究所** — 在仿生机器人、医疗机器人和空间机器人领域有突出贡献
    - **上海交通大学机器人研究所** — 研究方向涵盖医疗机器人、移动机器人和人机交互，与多家企业合作推动产学研结合
    - **西安交通大学** — 在智能机器人、人机交互和机器视觉领域有长期积累，拥有人工智能与机器人研究所
    - **南京理工大学** — 在特种机器人、排爆机器人和智能无人系统领域有特色研究
    - **国防科技大学** — 在无人系统、足式机器人和集群智能领域有深入研究，开发了多款军用和民用机器人系统
    - **浙江大学控制科学与工程学院** — 在移动机器人、飞行机器人和机器人视觉方面有重要研究，机器人足球队多次获得 RoboCup 冠军
    - **中国科学技术大学** — 在服务机器人和人工智能领域有重要研究成果，"可佳"（KeJia）系列服务机器人曾获多项国际竞赛冠军
    - **华中科技大学** — 在数字制造装备和工业机器人领域有深厚积累
    - **东南大学** — 在移动机器人和智能交通系统领域有长期研究
    - **同济大学** — 在自动驾驶和智能建造机器人领域有特色研究
    - **中国科学院自动化研究所** — 在脑科学与类脑智能、模式识别等领域有重要研究，为机器人智能提供理论基础
    - **中国科学院深圳先进技术研究院** — 在医疗机器人、仿生机器人和智能传感器领域有创新性研究
    - **北京理工大学** — 在仿人机器人和特种地面无人系统领域有重要研究，开发了"汇童"仿人机器人
    - **南开大学机器人与信息自动化研究所** — 在微纳操作机器人和机器人控制理论方面有特色研究
- 香港 (HK, China)
    - **香港科技大学** — 在自主系统和无人机领域有领先研究，沈劭劼教授团队在四足机器人方面有突出贡献
    - **香港中文大学** — T Stone Robotics Institute 在医疗机器人、无人机和多机器人系统领域有重要研究成果
    - **香港大学** — 机械工程系在机器人操作和柔性机器人领域有深入研究
- 澳门 (Macau, China)
    - **澳门大学** — 在智能系统和嵌入式系统研究方面有持续投入
